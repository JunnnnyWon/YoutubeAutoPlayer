# Linear - School Music Timer 사용 설명서

## 🎯 클라이언트 요구사항에 맞춘 새로운 UI

클라이언트가 제공한 PowerPoint 모형에 맞춰 완전히 새롭게 설계된 Linear 스타일 UI입니다.

### 📱 UI 구성 요소

#### 1. 상단 바 (Header)
- **🔴 Linear 로고**: 브랜드 아이덴티티
- **프로그램 ID**: #332E2B (고유 식별자)
- **실시간 시계**: 현재 시간 표시

#### 2. 왼쪽 사이드바 (Navigation)
- **⏰ Schedule**: 주간 스케줄 관리
- **📁 Playlists**: 플레이리스트 관리 (2개 메뉴)
- **⚙️ Settings**: 시스템 설정

#### 3. 중앙 메인 영역 (Schedule Grid)
- **주간 스케줄 그리드**: 월-일요일 × 6-22시
- **스케줄 슬롯**: 클릭하여 시간대별 설정
- **상태 표시**: 예약된 시간 오렌지색 표시
- **시간/남은시간**: 각 슬롯에 정보 표시

#### 4. 하단 컨트롤 (Control Buttons)  
- **시작**: 스케줄러 실행
- **중지**: 스케줄러 정지  
- **저장**: 설정 저장

#### 5. 오른쪽 패널 (Broadcast Log)
- **Recent Broadcast Log**: 실시간 로그
- **확대 가능**: 크기 조정 가능
- **실행 내역**: 타임스탬프와 함께 기록

## 🚀 사용 방법

### 1️⃣ 플레이리스트 설정

1. 왼쪽 사이드바에서 **"Playlists"** 클릭
2. **YouTube Playlist** 또는 **Local Audio Files** 선택
3. URL 입력 또는 파일 선택
4. **"Add"** 버튼으로 추가
5. **"Save & Close"**로 저장

**지원 형식:**
- YouTube: 플레이리스트 URL 또는 개별 비디오 URL
- 로컬: MP3, WAV, OGG, M4A, FLAC

### 2️⃣ 스케줄 설정

1. 중앙 그리드에서 원하는 **시간 슬롯 클릭**
2. 스케줄 다이얼로그에서:
   - **플레이리스트 선택**
   - **종료 시간 설정**
3. **"Save Schedule"** 클릭
4. 설정된 시간이 **오렌지색**으로 표시됨

### 3️⃣ 스케줄 실행

1. 하단의 **"시작"** 버튼 클릭
2. 버튼이 **"실행중..."**으로 변경
3. 오른쪽 로그에서 **실시간 상태 확인**
4. 필요시 **"중지"** 버튼으로 정지

### 4️⃣ 실시간 모니터링

**오른쪽 Broadcast Log에서 확인:**
- ⚪ Live Stream Started (시작 시간)
- ⚪ Segment Transition (전환)  
- ⚪ Audio Alert (알림)
- ⚪ Scheduler started/stopped

## 🔧 고급 기능

### 🛡️ 통합 미디어 컨트롤러
- **충돌 방지**: YouTube와 로컬 음원 동시 재생 방지
- **안전한 전환**: 미디어 간 부드러운 전환
- **상태 모니터링**: 실시간 재생 상태 추적

### ⏰ 요일별 스케줄링
- **주간 스케줄**: 월-일요일 개별 설정
- **시간대별**: 6시-22시 시간 단위 설정
- **자동 반복**: 매주 동일한 패턴 실행

### 💾 자동 저장/복원
- **설정 자동 저장**: config.ini 파일에 저장
- **프로그램 재시작시 복원**: 기존 스케줄 유지
- **백업 기능**: gui_backup.py로 이전 버전 보관

## ⚠️ 주의사항

### 시스템 요구사항
- **Python 3.7+** 필수
- **PyQt6**: GUI 프레임워크
- **Selenium**: YouTube 자동화
- **pygame**: 로컬 음원 재생

### 네트워크 연결
- YouTube 재생시 **안정적인 인터넷 연결** 필요
- 로컬 음원은 **오프라인 재생** 가능

### 파일 경로
- 로컬 음원 파일 경로가 **변경되지 않도록 주의**
- 상대 경로보다는 **절대 경로** 권장

## 🔄 문제 해결

### GUI 실행 오류
```bash
# 의존성 재설치
pip install PyQt6 selenium pygame

# GUI 실행
python gui.py
```

### YouTube 재생 실패
- **인터넷 연결** 확인
- **YouTube URL 유효성** 검증
- **크롬 브라우저** 업데이트

### 로컬 음원 재생 실패
- **파일 경로** 확인
- **파일 형식** 지원 여부 확인 (MP3, WAV, OGG, M4A, FLAC)
- **pygame 설치** 확인
